Welcome to Drug Test Importer

This tool scrapes CRL and i3 Screen, normalizes the results, and splits them into:

  • Worklist – raw records pending review
  • Review – incomplete records you can fill in

To run the pipeline, use your CLI:

python run.py

Or, in dry-run mode:
python run.py --dry-run

Once you’ve run it, click one of the links above to see your data.
